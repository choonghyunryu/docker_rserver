# R 서버를 위한 docker 이미지

## R 서버 환경

1. Web Server
    - Apache 2
    - 정적 웹 페이지 퍼블리싱
2. RStudio Server
    - 서버에 접속하여 R 분석
3. Shiny Server
    - 개발한 Shiny 앱 배포
4. 퍼블리싱 관련 작업
    - tex
    - Quarto
5. 한글 NLP
    - 은접한닢 형태소분석기
    - Rmecab
6. google chrome
    - pagedown을 이용한 문서의 pdf 출력
7. 한글 폰트 설치
    - 나눔 폰트 시리즈
